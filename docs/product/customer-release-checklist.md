Используется при онбординге студии/сети в Podocare. Два трека публикации — см. `docs/launch-backlog.md` §3.

**Ответственный со стороны заказчика:** _______________  
**Дата старта онбординга:** _______________  
**Трек публикации:** ☐ A (публикуем мы) ☐ B (публикует заказчик)

---

## 1. Юридическое лицо и договор

- [ ] Реквизиты юрлица / ИП для B2B-договора и счетов
- [ ] Подписан договор лицензии + приложение «Тариф» (`b2b-tariff.md`)
- [ ] Определён **оператор ПДн** клиентов (обычно заказчик) и URL **политики конфиденциальности** (HTTPS)
- [ ] Тексты согласий в приложении согласованы с юристом заказчика (шаблоны Podocare — стартовая точка)

---

## 2. Tenant в платформе (даёт Podocare после договора)

---

## 5. Трек A — публикуем мы за заказчика

- [ ] Доступ к **Apple Developer** (роль Admin / App Manager) или приглашение в наш team по согласованию
- [ ] Доступ к **Google Play Console** (роль Release manager) или создание приложения заказчиком + передача нам
- [ ] Оплачен Apple Developer Program (заказчик)
- [ ] Создана запись приложения в консолях (или поручение нам)
- [ ] Подпись / доверенность на действия в сторах (если требует юрист)
- [ ] Тестовый аккаунт для ревью Apple (логин/пароль) — если нужен вход в приложение

---

## 6. Трек B — публикует заказчик

- [ ] Получен **release kit** от Podocare: `.aab` (Android) / инструкция TestFlight (iOS)
- [ ] Получены `networkId`, API URL, версия сборки
- [ ] Заказчик загрузил билд и прошёл ревью самостоятельно
- [ ] Заказчик сообщил ссылки на опубликованные listing (URL App Store / Play)

---

## 7. Эквайринг и уведомления (при включении в тариф)

- [ ] Ключи ЮKassa / Т‑Банк (или решение использовать только офлайн-оплату на старте)
- [ ] Webhook URL согласован с prod API (HTTPS)
- [ ] SMS-шлюз / отправитель (если SMS в prod) — ключи заказчика или договорённость с Podocare

---

## 8. Приёмка

- [ ] Сборка установлена на тестовое устройство заказчика
- [ ] Вход персонала в админку, тестовая запись
- [ ] Клиентский сценарий: установка → OTP → запись (минимум)
- [ ] Подпись акта приёмки онбординга
