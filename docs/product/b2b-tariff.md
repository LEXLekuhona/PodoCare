# Тариф Podocare (B2B SaaS) — v0.3 (утверждён)

**Статус:** **утверждён** (вариант «базовый», 2026-06). Использовать в оферте и приложении к B2B-договору.  
**Модель:** подписка студии/сети; мобильное приложение white-label; публикация в сторах — [трек A или B](customer-release-checklist.md).

> Цены ориентированы на **независимую студию** с выручкой от ~500 тыс. ₽/мес и выше.  
> White-label приложение + контент-воронка — заметно выше класса «только запись» (DIKIDI / YClients ~2–6 тыс. ₽/мес).

---

## Для кого

Независимые подологические студии и небольшие сети (1–5 точек), которым нужны:

- записи, напоминания, кабинет специалиста;
- контент-воронка, квиз, программы, оплаты;
- **брендированное** мобильное приложение для клиентов.

**Оператор ПДн клиентов студии** — студия (заказчик). **Podocare** — лицензиар ПО и обработчик по поручению (см. договор).

## Разовые услуги (онбординг)

| Услуга | Трек A (публикуем мы) | Трек B (публикует заказчик) |
|--------|------------------------|-----------------------------|
| **Разовый онбординг** | **49 900 ₽** — сбор материалов, white-label сборка, загрузка в **App Store + Google Play**, 1 итерация правок по отклонению стора | **24 900 ₽** — сборка, release kit (`.aab` + TestFlight), инструкция, 1 час созвона |
| **Повторная публикация** (ребрендинг, смена bundle id) | **19 900 ₽** / случай | **9 900 ₽** (консультация + новая сборка) |
| **Доп. точка** (только админка/API, без нового приложения) | **0 ₽** в рамках тарифа «Сеть» | то же |

**Не входит в онбординг:** фотосессия для стора, копирайтинг карточки, юридическая экспертиза текстов — по отдельной смете или силами заказчика.

## Trial и пилот

| Режим | Условия |
|-------|---------|
| **Пилот 60 дней** | 0 ₽, 1 студия, до 3 специалистов; онбординг трек B **24 900 ₽** (или трек A со скидкой **39 900 ₽** для первых 3 клиентов) |

## Обоснование цифр (для внутреннего согласования)

| Решение | Логика |
|---------|--------|
| **19 900 ₽ Старт** | ~1–3% выручки типичной студии; выше «голой записи», но есть **своё приложение** + воронка + программы |
| **44 900 ₽ Сеть (3 точки)** | ~15 000 ₽/точка — скидка за объём; один white-label app на сеть |
| **Онбординг 49 900 / 24 900** | 8–16 ч ручной работы (дизайн-чек, EAS, сторы, QA); трек A дороже из‑за ответственности за загрузку |
| **Пилот 0 ₽** | Снижает барьер входа; монетизация с онбординга и подписки после 60 дней |
| **Лимиты специалистов** | Защита от злоупотребления одним tenant; крупнее — индивидуальный договор |

## Не входит в подписку

- Индивидуальная разработка фич под одного заказчика.
- Юридические тексты B2C (политика ПДн, оферта клиентам студии) — заказчик + юрист; Podocare даёт шаблоны.
- Маркетинг, ASO, ведение карточки стора после онбординга.
- Гарантия прохождения ревью при нарушении правил стора или неполных материалах заказчика.
